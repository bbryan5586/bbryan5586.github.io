Vulnerability Database (CVEs)

A breakdown of notable vulnerabilities that target either corporate entities, or everyday internet users like you. More of these are discovered every single day.

Consumer Risks (Personal Devices)

These vulnerabilities target everyday technology like smartphones, laptops, and web browsers. They often rely on users clicking a bad link or opening a file.

CVE ID Software Severity (CVSS) Impact on You
CVE-2023-38831 WinRAR 7.8 (High) Hackers can hide malware inside ZIP files that look like harmless images.
CVE-2023-41064 Apple iOS & macOS 9.8 (Critical) Zero-click exploit allowing spyware to be installed via iMessage images.
CVE-2023-4863 Google Chrome / WebP 8.8 (High) Opening a malicious image on a website could instantly compromise your browser.
CVE-2024-21413 Microsoft Outlook 9.8 (Critical) Critical bug allows code execution just by previewing an email.
CVE-2024-0519 Google Chrome V8 8.8 (High) Attackers can crash your browser or run code by luring you to a bad website.
CVE-2024-30051 Windows 10/11 DWM 7.8 (High) Malware already on your PC can use this to gain full Administrator control.
CVE-2023-23397 Microsoft Outlook 9.8 (Critical) Zero-click exploit that steals your password hash just by receiving an email.
CVE-2024-4671 Google Chrome 8.8 (High) Use-after-free bug allowing hackers to take over your browser session.
CVE-2024-38021 Microsoft Office 8.8 (High) Opening a trusted-looking Office document can run malicious code.
CVE-2024-27815 Apple iOS & macOS 7.8 (High) Malicious apps can crash the system or write data to kernel memory.
