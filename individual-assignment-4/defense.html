Defense Strategies

Core Security Concepts

  • Zero Trust: "Never trust, always verify." Assume breach.
  • MFA (Multi-Factor Authentication): Using a password plus a phone code or app.
  • EDR: Endpoint Detection and Response tools to monitor computers.
  • Backups: Keeping offline copies of data to recover from Ransomware.

Red Teaming

Hiring ethical hackers to try and break into your system to find weak spots before the bad guys do.

Segmentation

Splitting a network into smaller parts. If one part gets hacked, the others stay safe.

Encryption

Scrambling data so that if it is stolen, it cannot be read without a key.
